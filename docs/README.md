# Confidant AI Website

This directory contains the static website for Confidant AI, deployed to GitHub Pages.

## Structure

- `index.html` - Main landing page with all CSS and JavaScript embedded

## Deployment

The website is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the main branch.

Visit: http://2796gaurav.github.io/confidantai

## Local Development

To view the website locally:

1. Open `index.html` in a web browser
2. Or use a local server:
   ```bash
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```
