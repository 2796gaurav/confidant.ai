Onboarding

Up and running
in minutes.

Four steps from download to your first conversation. No accounts, no subscriptions, no cloud setup required.

1
Download & Install
Download the APK from GitHub Releases. Enable installation from unknown sources in Android Settings and install.
2
Complete Setup
Enter your interests, connect your Telegram bot token, configure sleep mode hours, and grant necessary permissions.
3
Download Model
From the Dashboard, tap "Download Model Now." The AI model is ~2 GB. WiFi connection strongly recommended.
4
Start Chatting
Tap "Start Server" on the Dashboard, then open Telegram and begin your first conversation with your AI companion.
03
Privacy Architecture

Your data never
leaves the device.

Privacy isn't a feature — it's the foundation every decision is built on.

Confidant AI was designed with a single architectural constraint: no personal data, no conversation history, no inferences about you should ever be transmitted to a third-party server. The only external network calls the app makes are your deliberate web searches via DuckDuckGo.

The full source code is publicly available on GitHub. Read it, audit it, compile it yourself.

100% Local Storage
All conversation and preference data lives only on your device. No syncing to external servers.
No Personal Identifiers
We do not collect names, gender, device IDs, or any personally identifiable information.
Open Source & Auditable
Every line of code is public. Trust the engineering, not our word.
Zero Analytics & Telemetry
No crash reporters, no usage analytics, no A/B testing frameworks. Nothing.
Encrypted Storage (Android 9+)
All local data is encrypted at rest on Android 9 and above. Never readable from other apps.
Web Search Isolation
Only explicit web search queries leave the device. No conversation context is ever attached.
04
Installation

Step-by-step
install guide.

Android requires a few extra taps to install apps outside the Play Store. Here's exactly what to do.

01
Download the APK
Click the download button and save confidant-ai-v1.0.0-1-release.apk to your Android device. All releases are also available on GitHub.
02
Enable Unknown Sources
Settings → Security → Install Unknown Apps. Enable for your browser or file manager. This allows installing apps outside the Play Store.
03
Install the APK
Open the downloaded APK from notifications or your file manager. Tap "Install" and grant all requested permissions.
04
Complete Onboarding
Launch Confidant AI. Enter your interests, connect your Telegram bot token, set sleep mode hours, and grant notification access.
05
Download the AI Model
From the Dashboard, tap "Download Model Now." The model is ~2 GB. Connect to WiFi before starting to avoid mobile data charges.
06
Start Chatting
Once the model is downloaded, tap "Start Server" on the Dashboard. Open Telegram, find your bot, and send your first message.
